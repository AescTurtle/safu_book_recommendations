# 📚 Книги по настроению

Веб-приложение, которое подбирает книги под настроение пользователя.
Учебный проект · САФУ · группа 152419.

Пользователь выбирает своё эмоциональное состояние — приложение предлагает
пять книг с подходящими тегами. Можно листать каталог, искать книги, оценивать
их и оставлять отзывы. Администратор управляет каталогом и выгружает данные в CSV.

## Возможности

- Подбор книг по настроению (8 настроений, 52 книги)
- Каталог с поиском и фильтром по настроению
- Регистрация, вход, выход (Flask-Login, хеширование паролей)
- Оценки книг (1–5) и отзывы
- Личный кабинет: история рекомендаций и оценок
- Админка: CRUD книг, статистика, экспорт CSV (книги и рекомендации)
- Защита форм от CSRF (Flask-WTF)

## Технологии

- **Python 3.12**, **Flask** — веб-фреймворк (фабрика приложения `create_app()`)
- **Flask-SQLAlchemy** — ORM, **SQLite** — база данных
- **Flask-Login** — сессии и авторизация
- **Flask-WTF** — защита от CSRF
- **Bootstrap 5** — вёрстка (через CDN)
- **Gunicorn** — продакшн-сервер
- **pytest** — тесты
- **Docker** / **docker compose** — контейнеризация

## Быстрый старт (локально)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

python seed.py          # создать БД и загрузить данные (8 настроений, 52 книги)
flask run               # http://127.0.0.1:5000
```

Администратор по умолчанию: **admin / admin** (пароль задаётся через `ADMIN_PASSWORD`).

## Запуск в Docker

```bash
docker compose up --build
```

Приложение доступно на http://localhost:5000. База данных хранится в томе `db_data`
и сидируется при первом запуске (см. `entrypoint.sh`).

## Тесты

```bash
pip install -r requirements-dev.txt
pytest
```

## Переменные окружения

| Переменная       | Назначение                          | По умолчанию              |
|------------------|-------------------------------------|---------------------------|
| `SECRET_KEY`     | ключ подписи сессий                 | небезопасный dev-ключ     |
| `DATABASE_URL`   | строка подключения к БД             | `sqlite:///app.db`        |
| `ADMIN_PASSWORD` | пароль администратора при сидинге   | `admin`                   |
| `FLASK_DEBUG`    | режим отладки (`1` — включить)      | `0`                       |

## Структура проекта

```
app.py             — фабрика приложения и маршруты
models.py          — модели БД (User, Mood, Book, Rating, Recommendation)
seed.py            — инициализация БД и загрузка данных из data/*.json
data/              — moods.json (8 настроений), books.json (52 книги)
templates/         — Jinja2-шаблоны (base, index, каталог, админка, ошибки)
static/style.css   — стили
tests/             — pytest-тесты (auth, app, admin)
conftest.py        — фикстуры pytest
Dockerfile, docker-compose.yml, entrypoint.sh — деплой
```

## Маршруты

| Метод | Путь                          | Доступ   | Назначение                  |
|-------|-------------------------------|----------|-----------------------------|
| GET   | `/`                           | все      | главная, выбор настроения   |
| POST  | `/recommend`                  | вход     | подбор 5 книг по настроению |
| GET   | `/books`                      | все      | каталог (поиск, фильтр)     |
| GET   | `/books/<id>`                 | все      | страница книги, отзывы      |
| POST  | `/books/<id>/rate`            | вход     | оценить книгу               |
| GET   | `/register`, `/login`         | гость    | регистрация / вход          |
| POST  | `/logout`                     | вход     | выход                       |
| GET   | `/dashboard`                  | вход     | личный кабинет              |
| GET   | `/admin`                      | админ    | статистика                  |
| *     | `/admin/books...`             | админ    | CRUD книг                   |
| GET   | `/admin/export/*.csv`         | админ    | выгрузка CSV                |
