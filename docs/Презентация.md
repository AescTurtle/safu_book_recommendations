# 📚 Книги по настроению

### Веб-сервис подбора книг по эмоциональному состоянию

## Проблема и актуальность

- Литературы очень много - читателю трудно выбрать книгу **«под настроение»**.
- Каталоги сортируют по жанру и рейтингу, но **не по эмоциональному запросу**.
- Подход «что почитать, когда грустно / тревожно / хочется вдохновиться» востребован,
  но слабо представлен в русскоязычных сервисах.

> Цель - снизить «паралич выбора»: пользователь указывает настроение и получает
> компактную подборку подходящих книг.

---

## Цель и задачи

**Цель:** разработать веб-приложение, подбирающее книги по текущему настроению.

**Задачи:**
1. Проанализировать предметную область и аналоги.
2. Спроектировать ролевую модель и базу данных.
3. Обосновать выбор средств реализации.
4. Реализовать приложение (подбор, оценки, админка, отчёты).
5. Провести тестирование.
6. Подготовить документацию и презентацию.

## Анализ аналогов

| Сервис | Подбор по настроению | Ограничения |
|---|---|---|
| LiveLib | частично (подборки) | неявный, через поиск |
| Goodreads | нет | англоязычный |
| MyBook / Литрес | нет | рекомендации по жанрам/покупкам |
| Книжные Telegram-боты | иногда | узкий каталог, нет ролей |

**Вывод:** есть ниша для простого русскоязычного сервиса с подбором именно по настроению.

## Роли пользователей

Три иерархичные роли:

- **Гость** - просмотр каталога и карточек книг (только чтение).
- **Пользователь** - + подбор по настроению, оценки, личный кабинет.
- **Администратор** - + статистика, CRUD книг, экспорт отчётов CSV.

> Требование задания «≥2 типов пользователей + незарегистрированный» выполнено.

## Модель данных (6 таблиц)

- **users** - пользователи (логин, хеш пароля, роль)
- **moods** - настроения (код, название, эмодзи)
- **books** - книги (название, автор, год, описание)
- **book_moods** - связь книга ↔ настроение (M:N)
- **recommendations** - история подборов
- **ratings** - оценки 1–5 с комментарием (одна на книгу от пользователя)

---

## Функция: подбор по настроению

1. Пользователь выбирает одно из **8 настроений** на главной.
2. Система возвращает **5 случайных** подходящих книг.
3. Каждый запрос сохраняется в **историю рекомендаций**.

🟢 Грустно · Тревожно · Радостно · Скучно · Ностальгия · Подъём сил · Любопытно · Романтично

---

## Каталог, оценки, личный кабинет

- **Каталог** - поиск по названию/автору + фильтр по настроению.
- **Карточка книги** - описание, настроения, средняя оценка, отзывы.
- **Оценки** - 1–5 с комментарием; повторная оценка обновляет прежнюю.
- **Личный кабинет** - история рекомендаций и собственные оценки.

---

## Админ-панель и отчёты

- **Статистика** - счётчики пользователей, книг, настроений, рекомендаций, оценок.
- **CRUD книг** - добавление, редактирование, удаление, привязка настроений.
- **Экспорт отчётов CSV:**
  - каталог книг,
  - история рекомендаций.

🔒 Доступ только для роли «администратор» (`admin_required`).

## Тестирование

**Автотесты (pytest): 26 тестов, все проходят.**
